Stiahnutie a spustenie

1. Stiahnut baliky:
cd ~/catkin_ws/src
git clone https://github.com/MichalDobis/Universal_Robots_ROS_Driver # pouzit branch master
git clone https://github.com/MichalDobis/universal_robot # pouzit branch melodic-devel
cd universal_robot
git checkout melodic-devel

2. Skompilovat
cd ~/catkin_ws
catkin_make

3. Nakonfigurovat robot
https://github.com/MichalDobis/Universal_Robots_ROS_Driver/blob/master/ur_robot_driver/doc/install_urcap_e_series.md

4. Spustanie 
roslaunch ur5_e_moveit_config demo.launch # Spusti moveit config robota aj s modelom
roslaunch ur_robot_driver ur5e_bringup.launch # Spusti komunikaciu s robotom aj s kontrolermi


Dalsie dolezite veci, ktore som spravil a su commitnute. Cize staci iba git clone
